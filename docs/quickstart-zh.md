# 杀死苏丹 Online 新手开局指南（中文版）

这份指南给第一次接触项目的玩家使用，目标是 5 分钟内开房并开始游戏。

## 1. 先准备环境

- 安装 Node.js（建议 20+，至少 18+）
- 打开 PowerShell（Windows）或终端（macOS/Linux）
- 进入项目根目录（就是包含 `package.json` 的目录）

## 2. 一键启动项目

在项目根目录依次执行：

```bash
npm install
npm run dev
```

启动成功后，你会看到两个地址：

- 前端页面：`http://localhost:5173`
- 游戏服务：`http://localhost:3000`

如果只看到前端地址也没关系，直接打开 `http://localhost:5173` 即可。

## 3. 第一次进入怎么玩

1. 打开浏览器访问 `http://localhost:5173`
2. 输入你的昵称（`Player Name`）
3. 主持人点击 `Create Room` 创建房间
4. 记下页面显示的房间号（如 `AB12CD`）
5. 其他玩家在自己的页面输入同一个房间号，点击 `Join Room`
6. 所有人进入后，各自点击 `Ready`
7. 房主点击 `Start Game`

到这里就可以正式开始了。

## 4. 一台电脑怎么多人试玩

如果你只有一台电脑，也可以快速试玩：

- 方法 A：同一个浏览器开多个无痕窗口
- 方法 B：用不同浏览器（Chrome / Edge / Firefox）分别开页面
- 每个窗口用不同昵称，加入同一个房间

## 5. 回合内可执行动作（当前版本）

轮到你时，可以执行一个动作：

- `Peek`：偷看目标玩家暗牌（只有你自己看到结果）
- `Swap`：和目标玩家交换暗牌（明牌不能换）
- `Swap Center`：和中间牌交换
- `Reveal`：公开自己的角色并触发角色技能

提示：

- 页面里会显示当前是谁的回合（`Turn`）
- 不是你的回合时按钮会禁用
- 行动日志会显示“做了什么”，但不会泄露暗牌细节

## 6. 新手常见问题

### Q1：页面打不开（5173 无响应）

- 确认 `npm run dev` 没有中断
- 看终端有没有报错
- 换个端口占用少的环境重试

### Q2：加入房间失败

- 房间号要完全一致（英文字母建议全大写）
- 游戏开始后只允许原玩家用 token 重连，不支持新玩家插入

### Q3：我点动作没反应

- 先确认是不是你的回合
- 是否被拘留（detained）
- 目标是否合法（例如不能对死亡玩家执行某些动作）

### Q4：断线了怎么办

- 直接刷新页面
- 客户端会尝试用本地 token 自动重连到原房间

## 7. 你可以这样给朋友快速发开局步骤

```text
1) 下载项目并在根目录执行 npm install
2) 执行 npm run dev
3) 打开 http://localhost:5173
4) 1 个人建房，其他人输房间号加入
5) 全员 Ready 后房主 Start Game
```

## 8. 当前版本说明（MVP）

- 已有：建房 / 加房 / 准备 / 开局 / 回合动作 / 私密信息隔离 / 断线重连
- 暂无：正式账号系统、匹配大厅、战绩榜、完整 UI 引导、生产级部署脚本

如果你是第一次带人试玩，建议先用 5 人局，在同一台机器多窗口演练 1 局流程，再开给真实玩家。
