# 刺杀苏丹王 KillSultan 新手开局指南（简体中文）

这份文档只做一件事：帮你从 0 到 1 启动游戏并开出第一局。

## 1. 准备环境（只需要一次）

1. 安装 Node.js（建议 20+，最低 18+）。
2. 打开终端（Windows 推荐 PowerShell）。
3. 进入项目根目录（能看到 `package.json` 的目录）。

## 2. 启动游戏（照抄命令）

```bash
npm install
npm run dev
```

启动后默认地址：

- 客户端：`http://localhost:5173`
- 服务端：`http://localhost:3000`
- 服务端健康检查：`http://localhost:3000/health`

看到 `health` 返回 `{"ok":true,...}` 就说明后端正常。

## 3. 开第一局（3 分钟）

1. 房主打开 `http://localhost:5173`，输入昵称，点击“创建房间”。
2. 其他玩家打开同一地址，输入昵称 + 房间号，点击“加入房间”。
3. 全员点击“准备”。
4. 房主点击“开始游戏”。

## 4. 一台电脑多人测试

你可以不找人，自己在一台电脑上测多人流程：

1. 使用多个无痕窗口，或者多个浏览器。
2. 每个窗口用不同昵称加入同一房间。
3. 按正常流程准备并开局。

## 5. 单窗口实验台（5-15 模拟玩家）

如果你要做压测或快速回归，打开：

`http://localhost:5173/?lab=1`

支持模拟 5-15 玩家、全流程动作、断线重连，以及高级调试事件。

## 6. 常见问题排查

### Q1：页面打不开

- 确认 `npm run dev` 还在运行（别把终端关掉）。
- 访问 `http://localhost:3000/health` 看后端是否存活。
- 访问 `http://localhost:5173` 看前端是否存活。

### Q2：提示 3000 / 5173 端口被占用

- 关闭之前已经启动的同类进程，再重试。
- 或改端口后再启动（例如设置 `PORT`、`VITE_SERVER_URL`）。

### Q3：加入房间失败

- 房间号要完全一致（建议全大写）。
- 游戏开始后不支持新玩家中途加入，只支持原玩家重连。

### Q4：动作按钮点不了

- 可能不是你的回合。
- 可能被拘留（需要跳过行动）。
- 目标可能不合法（比如已死亡）。

### Q5：断线后怎么回房间

- 刷新页面即可。
- 客户端会尝试用本地 token 自动重连。

## 7. 常用命令速查

- `npm run dev`：一键启动完整开发环境
- `npm run dev:server`：仅启动服务端
- `npm run dev:client`：仅启动客户端
- `npm run typecheck`：检查类型
- `npm run docs:sync`：同步文档到客户端 `public/docs`
