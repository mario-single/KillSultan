# 杀死苏丹 Online 新手开局指南（简体中文）

这份文档面向第一次接触项目的玩家，目标是从 0 到开局只用几分钟。

## 1. 环境准备

- 安装 Node.js（建议 20+，最低 18+）
- 打开 PowerShell（Windows）或终端（macOS / Linux）
- 进入项目根目录（包含 `package.json` 的目录）

## 2. 启动项目

在项目根目录执行：

```bash
npm install
npm run dev
```

启动后默认地址：

- 客户端：`http://localhost:5173`
- 服务端：`http://localhost:3000`

直接打开客户端地址即可开始。

## 3. 第一次开房流程

1. 在浏览器打开 `http://localhost:5173`
2. 输入昵称
3. 房主点击“创建房间”
4. 记下页面上的房间号
5. 其他玩家输入同一房间号并点击“加入房间”
6. 所有人点击“准备”
7. 房主点击“开始游戏”

完成以上步骤后，系统会发牌并进入第一轮。

## 4. 一台电脑如何多人测试

你可以用下面任一方式快速本地演练：

- 同一浏览器开多个无痕窗口
- 用不同浏览器同时打开页面（Chrome / Edge / Firefox）
- 每个窗口使用不同昵称，加入同一个房间

## 5. 回合可执行动作

轮到你时，每回合只能执行一个动作：

- 偷看：秘密查看目标玩家的身份牌
- 交换：与目标玩家交换暗牌（明牌不能交换）
- 换中间牌：与中间牌交换
- 公开：翻开自己的牌并触发角色技能

页面中会显示“当前行动者”，不是你的回合时按钮会自动禁用。

## 6. 常见问题

### Q1：打不开页面

- 确认 `npm run dev` 仍在运行
- 检查终端是否有报错
- 确认 `5173`、`3000` 端口没有被其他程序占用

### Q2：加入房间失败

- 房间号必须完全一致（建议全部大写）
- 游戏开始后，新玩家不能中途加入，只支持原玩家重连

### Q3：动作按钮无法点击

- 可能不是你的回合
- 你可能处于拘留状态
- 你选择的目标不合法（例如已死亡）

### Q4：断线后怎么回到对局

- 刷新页面即可
- 客户端会用本地 token 自动尝试重连原房间

## 7. 发给朋友的超短开局版

```text
1) 根目录执行 npm install
2) 执行 npm run dev
3) 打开 http://localhost:5173
4) 一人建房，其他人输房间号加入
5) 全员准备后房主开始
```
