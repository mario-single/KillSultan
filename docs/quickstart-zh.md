# 杀死苏丹 Online 新手开局指南（简体中文）

## 1. 环境准备

- 安装 Node.js（建议 20+，最低 18+）
- 打开终端（Windows 推荐 PowerShell）
- 进入项目根目录（包含 `package.json`）

## 2. 启动项目

```bash
npm install
npm run dev
```

默认地址：

- 客户端：`http://localhost:5173`
- 服务端：`http://localhost:3000`

## 3. 第一局怎么开

1. 打开 `http://localhost:5173`
2. 输入昵称
3. 房主点击“创建房间”
4. 其余玩家输入房间号点击“加入房间”
5. 全员点击“准备”
6. 房主点击“开始游戏”

## 4. 一台电脑多人测试

- 同一浏览器开多个无痕窗口
- 或使用不同浏览器分别打开页面
- 每个窗口使用不同昵称加入同一房间

## 5. 回合动作

每回合只能执行一个动作：

- 偷看
- 交换
- 换中间牌
- 公开身份并触发技能

## 6. 常见问题

### Q1：页面打不开

- 检查 `npm run dev` 是否仍在运行
- 检查 `5173` / `3000` 端口是否被占用

### Q2：加入房间失败

- 房间号必须完全一致（建议全大写）
- 游戏开始后不支持新玩家中途加入，仅支持原玩家重连

### Q3：动作按钮点不了

- 可能不是你的回合
- 可能被拘留
- 目标可能不合法（例如已死亡）

### Q4：断线后如何回到房间

- 刷新页面即可
- 客户端会自动尝试用本地 token 重连
